Toggl Report Viewer

v0.1.0 (Sep 8, 2016)

Allows you to view a toggl report in week view.

Usage

  1. Create an account or log into toggl.com
  2. Go to the app page
  3. Check your settings
    • First day of the week: Monday
    • (Unknown if the following settings are required)
    • Date format: MM/DD/YYYY
    • Time format: 24-hour
  4. Mouse over the "Reports" tab and select "Detailed" from the dropdown list
  5. Click the "Export" button and select "Download as CSV" from the dropdown list
  6. Use the file submission interface below to submit the csv file
